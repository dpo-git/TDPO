Что означают функции
  • comb(n, k) — без учёта порядка (сколько подмножеств размера k из n).
  • perm(n, k) — с учётом порядка (сколько упорядоченных наборов из k элементов без повторений).
Это два базовых «кирпичика» для задач: «выбрать группу» (comb) и «распределить места/роли» (perm).

# Часть B. Инструменты перебора: itertools

В комбинаторике мы считаем, сколько существует разных вариантов выбора элементов. В Python это удобно делать с помощью модуля itertools, который умеет перебирать варианты точно так же, как и мы в наших с Вами головах.

# Задания для закрепления. Совет учеников и роли в нём

В классе 12 человек. Сколько способов выбрать совет из 3 человек и назначить председателя? Решите двумя способами: (1) формулы comb/perm, (2) перебор itertools.

Поясните словами, почему формула работает так же, как перебор.

⬇️ Напишите свой код в ячейке ниже.
In [7]
💡 Подсказка
  • Сначала состав совета (comb), затем выбор председателя из этих троих (×3).
  • В переборе: сначала it.combinations(range(12), 3), внутри — перебор председателя.
🧩 Посмотреть правильный ответ 
from math import comb
import itertools as it

# 1) Формула: выбираем троих из 12 и назначаем одного председателем
ways_formula = comb(12, 3) * 3 # comb(12,3) составы * 3 возможных председателя

# 2) Перебор с itertools: комбинации по 3 и выбор председателя
students = list(range(12))                        # обозначим учеников числами 0..11
ways_enum = 0                                     # счётчик способов
for trio in it.combinations(students, 3): # все тройки (без порядка)
    for chair in trio:                            # выбираем председателя внутри тройки
        ways_enum += 1                            # один способ найден

print(ways_formula, ways_enum)

Смысловое сравнение
  • Формула comb(12, 3) * 3 отражает: «выбрать состав» × «назначить председателя».
  • Перебор даёт тот же результат, потому что он фактически делает то же самое, только явно перечисляя варианты.
